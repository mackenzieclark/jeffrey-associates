Jeffrey Associates

About this extension

Jeffrey Associates highlights social media accounts belonging to individuals named in the Epstein files as associates of Jeffrey Epstein. Highlighted accounts appear in amber on supported social networks.

Data source

The list of associates is curated from publicly available Epstein court documents and flight logs. The dataset is encoded in a bloom filter, which means:

  • No names are stored in plain text in the extension.
  • A small number of false positives (incorrect highlights) may occur.
  • There are no false negatives â€” if someone is on the list, they will always be highlighted.

Supported platforms

The extension highlights profiles and links on: Twitter/X, Facebook, YouTube, Reddit, Bluesky, Mastodon, Threads, Medium, Tumblr, Wikipedia, and search engine results.

Settings

You can adjust the highlight color from the Settings page.
